Fundamentos de Acabados Superficiales

Proceso de Rectificado

Granulometría y Textura

Muela fina (120-240): Ra ≤ 0,2 μm (N4-N5)
Muela media (60-100): Ra 0,4-0,8 μm (N6)

Variables Críticas

Profundidad (ae): < 10 μm
Velocidad alimentación (vf): 5-20 mm/min

Buenas Prácticas

  • Balance de muela para reducir vibraciones
  • Control de temperatura con refrigerante

Proceso de Pulido y Lapeado

Pulido

Abrasivos: Pastas diamantadas
Ra: 0,01-0,1 μm
Velocidad: 1000-3000 rpm
Fuerza: Homogénea

Lapeado

Método: Superficies planas + abrasivo
Ra: < 0,02 μm
Presión: < 1 MPa

Aplicaciones

  • Óptica de precisión
  • Hidráulica de alta precisión

Proceso de Chorreado y Shot Peening

Proceso

Proyección de granalla (esferas metálicas) sobre la superficie

Efectos

Endurecimiento superficial: Mejora resistencia a fatiga
Ra final: 0,8-3 μm

Parámetros Críticos

Diámetro granalla: 0,2-1 mm
Presión: 4-7 bar
Tiempo: Controlado

Proceso de Recubrimientos Funcionales

Tipos de Revestimientos

  • Cromo duro
  • Nitruro de titanio (TiN)
  • DLC (Diamond-Like Carbon)

Beneficios

Reducción Ra: Hasta 0,05 μm
Propiedades tribológicas: Menos fricción y desgaste

Parámetros de Aplicación

Grosor: 1-10 μm
Temperatura DLC: < 200°C
Compatibilidad: Recubrimiento-sustrato
